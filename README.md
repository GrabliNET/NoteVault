# 📝 NoteVault — Приложение для управления заметками

Приложение для создания, организации и хранения персональных заметок с поддержкой объединения в книги, экспорта в PDF и системой ярлыков для быстрого поиска.

---

## 📋 Описание проекта

**NoteVault** — настольное приложение для ведения заметок. Пользователь может создавать заметки, группировать их в тематические книги, назначать ярлыки и экспортировать содержимое в PDF. Проект не претендует на замену Obsidian — цель сделать простой, удобный инструмент без лишних зависимостей.

---

## 🧩 Модули проекта

### 1. Модуль создания и редактирования заметок
Предоставляет интерфейс для создания новой заметки, редактирования её содержимого и установки заголовка. Поддерживает базовое форматирование текста (жирный, курсив, заголовки). Каждая заметка хранит дату создания и последнего изменения.

### 2. Модуль хранения заметок
Отвечает за сохранение и загрузку заметок на диск (локальное хранилище или база данных). Обеспечивает целостность данных и автосохранение при редактировании.

### 3. Модуль книг (коллекций)
Позволяет объединять заметки в именованные книги — тематические группы. Пользователь может создавать, переименовывать и удалять книги, перемещать заметки между ними. Одна заметка может входить только в одну книгу.

### 4. Модуль ярлыков (тегов)
Реализует систему ярлыков: пользователь назначает одной заметке произвольное количество текстовых тегов. Ярлыки хранятся глобально и доступны для выбора при редактировании любой заметки.

### 5. Модуль поиска
Обеспечивает быстрый поиск по всему хранилищу: по тексту заметки, по заголовку, а также фильтрацию по ярлыкам и книгам. Результаты отображаются в режиме реального времени по мере ввода запроса.

### 6. Модуль экспорта в PDF
Формирует PDF-документ из одной заметки или из всей книги. PDF содержит заголовок, дату, содержимое и список ярлыков. Поддерживает базовое оформление (шрифт, поля, нумерация страниц при экспорте книги).

### 7. Модуль пользовательского интерфейса
Реализует основной экран приложения: боковая панель со списком книг и ярлыков, центральная область для списка заметок, правая панель для просмотра и редактирования. Обеспечивает навигацию и переключение между разделами.

### 8. Модуль настроек
Позволяет пользователю задать параметры приложения: тему оформления (светлая/тёмная), путь к директории хранения данных, настройки экспорта PDF (шрифт, размер страницы).

### 9. Модуль управления ярлыками
Предоставляет отдельный экран для просмотра всех существующих ярлыков, их переименования и удаления. При удалении ярлыка он снимается со всех заметок, к которым был привязан.

### 10. Модуль статистики хранилища
Отображает сводную информацию: количество заметок, количество книг, наиболее используемые ярлыки. Помогает пользователю ориентироваться в объёме накопленного контента.
