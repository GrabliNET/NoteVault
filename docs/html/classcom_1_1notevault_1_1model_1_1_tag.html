Подробное описание

Тег (ярлык) для быстрой фильтрации заметок.

Теги хранятся в таблице tags БД с ограничением UNIQUE COLLATE NOCASE (регистронезависимая уникальность). Связь с заметками реализована через промежуточную таблицу note_tags (many-to-many).

В YAML front-matter заметки теги записываются как список:

tags:
- java
- tutorial

Физически заметки с одним тегом размещаются в папке с именем тега, что позволяет ориентироваться в хранилище без приложения.
