Подробное описание

Прикреплённый файл, связанный с конкретной заметкой.

Вложения хранятся физически в папке _attachments/<note-slug>/ внутри vault, а в БД (таблица attachments) хранится только ссылка на файл — имя и относительный путь. Сам бинарный контент в БД не попадает.

Ссылка в Markdown

В тексте заметки вложение вставляется синтаксисом Obsidian-style:

![[photo.png]]
![[document.pdf]]

При рендере превью com.notevault.service.MarkdownRenderer подставляет абсолютный URI файла в тег <img> или <a>.

Удаление

При удалении заметки связанные записи в таблице attachments каскадно удаляются (FK ON DELETE CASCADE). Физические файлы удаляются отдельно в com.notevault.service.VaultService#deleteNote.
