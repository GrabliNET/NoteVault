Подробное описание

Точка входа приложения NoteVault.

Инициализирует Look & Feel (FlatLaf macOS-стиль), применяет тему (тёмная / светлая) из пользовательских настроек, затем запускает главное окно com.notevault.ui.MainWindow в потоке Event Dispatch Thread.

Порядок инициализации:

  1. Выбор темы (из Preferences или автодетектирование).
  2. Настройка UIManager (скругления, ширина скроллбара и т.д.).
  3. Инициализация DatabaseManager (без открытия конкретного vault).
  4. Создание и отображение MainWindow.
